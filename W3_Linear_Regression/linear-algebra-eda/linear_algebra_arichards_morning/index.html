Linear algebra¶

Within data science and perhaps at its core is the field of Machine learning.

  • Supervised learning - learn a mapping from inputs \(x\) to outputs \(y\) (given \(\mathcal{D}\))
  • Unsuperviesed learning - given only \(x\) the goal is to find interesting patterns

Note

\(x\) is often stored in an \(N x D\) matrix. Linear algebra is the branch of mathematics that works directly with matrices and it turns out that it is really useful for solving systems of linear equations like those from linear regression.

Before we jump in lets make sure we have a few other definations in mind

  • Scaler - a single real number (ints, floats etc)
  • Vector - a 1D array of numbers (real, binary, integer etc)
  • Matrix - a 2D array of numbers
  • Tensor - a tensor is similar to that of a matrix except that it may have an arbitrary number of indices
